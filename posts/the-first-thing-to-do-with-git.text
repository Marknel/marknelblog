# The first thing to do with Git

 - published: 2009-11-26 10:17
 - tags: git,

When you install Git, the first thing you should be doing is setting up your user details.

    git config --global user.name "Your name"  
    git config --global user.email "you@example.com"

Obviously change these values for your own information.

Next, you might want to have a look at [getting some pretty colours for Git](/how-to-colours-in-git).
